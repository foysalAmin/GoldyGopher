# GoldyGopher
A simple web application built with ASP.NET Core Razor Pages that loops through to find any value that is a multiple of three print 
“Goldy” instead of the number and for multiples of seven print “Gopher” instead.  For numbers 
that are multiples of both three and seven print “Goldy Gopher”.  

## Prerequisites
- Visual Studio 2022
- .NET 8.0 SDK

## Running the application

1. Clone the repository
   git clone https://github.com/foysalAmin/GoldyGopher
2. Navigate to the project directory
   cd GoldyGopher
3. Restore the project dependencies
   dotnet restore
4. Build the project
   dotnet build
5. Run the project
   dotnet run
